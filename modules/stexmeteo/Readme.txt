--------------------------
Note per l'installazione :
--------------------------

- Copiare il contenuto dell'archivio "stEX-Meteo 1.00" nella cartella modules di E-Xoopport.

- Installare il modulo "Meteo in Italia" dal Menu di Amministrazione.

- Rendere visibile il blocco "stEX-Meteo 1.00" dal Menu di Amministrazione.

- Attivare i diritti di accesso e/o visualizzazione riguardanti "Meteo in Italia" per i Gruppi dal Menu di Amministrazione.


--------
Cos'è? :
--------

Il modulo "stEX-Meteo" è un add-on per E-Xoopport che permette di avere sul proprio portale la situazione e le previsioni meteo per l'Italia in generale e per ogni singola provincia.
Il servizio meteo è offerto da IlMeteo.it (http://www.ilmeteo.it/) (un ringraziamento particolare ad Antonio Sanò).

Installation notes :
--------------------


- Copy "stEX-Meteo 1.00" content inside of E-Xoopport's Modules directory.

- Install "Meteo in Italia" Module from Admin Menu.

- Make the "Weather stEX-Meteo 1.00" Block visible from Admin Menu.

- Activate access and/or visualization rights about "Meteo in Italia" for your Groups from Admin Menu.


--------------
What's this? :
--------------

"stEX-Meteo" module is an E-Xoopport's add-on that gives you the possibility to check the italian forecasts weather.
Forecasts are edited by IlMeteo,it (http://www.ilmeteo.it/) (a particular thanks it goes to Antonio Sanò).
